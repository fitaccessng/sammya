    Import Staff from Excel

    Upload an Excel file to bulk import staff records with personal information, employment details, compensation, and deductions

    Upload Excel File
    A descriptive name to identify this import batch
    Supported formats: .xlsx, .xls (Maximum 10MB)
    ✓ Required Column Headers (MUST include these exactly):
    • first_name - Staff first name (e.g., "John")
    • last_name - Staff last name (e.g., "Doe")
    • email - Email address (e.g., "john.doe@company.com") - Must be unique
    • basic_salary - Monthly basic salary (e.g., "5000000" or "5000000.00")
    ⚠️ Important: Your Excel file MUST have these 4 columns with these exact names. Column names are case-sensitive.
    Optional Column Headers:
    Personal Info:
    • phone_number
    • gender (Male/Female/Other)
    • date_of_birth (YYYY-MM-DD)
    • address
    • city
    • state
    • marital_status
    Employment:
    • employee_id
    • joining_date (YYYY-MM-DD)
    • department (HR/Finance/Procurement/QC/Projects/Cost Control)
    • position
    • role (admin/hr_manager/hr_staff/etc.)
    • employment_type (Full-time/Part-time/Contract/Intern)
    Compensation:
    • basic_salary - Monthly basic salary (e.g., 5000000.00)
    • allowances - Monthly allowances (e.g., 500000.00)
    Deductions:
    • deduction_type_1 - Type (e.g., Tax, Insurance, Loan)
    • deduction_amount_1 - Amount (e.g., 100000.00)
    • deduction_type_2, deduction_amount_2, etc.
    • Add multiple deductions using numbered suffix
    Emergency Contact(s) - Next of Kin:
    • nok_full_name - Full name *
    • nok_relationship - Relationship (Spouse/Parent/Sibling/Child/Friend/etc.) *
    • nok_phone - Phone number *
    • nok_email - Email address
    • nok_address - Residential address
    • nok_city - City
    • nok_state - State
    • nok_is_primary - Mark as primary (TRUE/FALSE)

    Need Help?
    How to prepare your Excel file:
    1. Create an Excel file (.xlsx or .xls) with staff information
    2. Use the column headers listed above (first row of the spreadsheet)
    3. Fill in data for each staff member in subsequent rows
    4. Ensure email addresses are unique (not used by existing staff)
    5. Use valid date formats: YYYY-MM-DD (e.g., 2026-02-02)
    6. For salary and deductions: use numeric values (e.g., 5000000.00)
    7. Upload the file here to preview before importing
    Salary & Deductions:
    • Basic Salary: Monthly base salary amount
    • Allowances: Optional monthly allowances
    • Multiple Deductions: Add as many deductions as needed:
      • Column: deduction_type_1 (e.g., "Tax")
      • Column: deduction_amount_1 (e.g., 100000.00)
      • Column: deduction_type_2, deduction_amount_2, etc.
    Emergency Contact (Next of Kin):
    • Full Name: Required - Complete name of emergency contact
    • Relationship: Required - Spouse, Parent, Sibling, Child, Friend, etc.
    • Phone: Required - Contact phone number
    • Email: Optional - Email address
    • Address: Optional - Residential address
    • City & State: Optional - City and state/province
    • Primary Contact: Use TRUE/FALSE to mark as primary emergency contact
    What happens after upload:
    1. System validates all data in the file
    2. Shows you a preview of valid and invalid records
    3. You can review and adjust role assignments
    4. Submit for admin approval when ready
    5. Admin reviews and imports the batch
    6. Staff accounts are created with default password based on their email
    7. Compensation, deductions, and emergency contacts are automatically linked
